# Mengimpor LSK ke MetaMask
Secara default, MetaMask tidak menampilkan token LSK yang dimiliki dalam akun wallet. 
Ini berarti pemegang LSK harus mengimpor token LSK secara manual ke dalam wallet di berbagai jaringan, agar dapat ditampilkan di wallet MetaMask.

:::note
Pastikan wallet Anda terhubung ke jaringan yang relevan, yang bisa berupa:
- **Mainnet Lisk**
- **Testnet Lisk Sepolia**, atau
- **Ethereum**

Untuk informasi lebih lanjut, lihat informasi terkait dalam panduan [Cara Menghubungkan Lisk ke Wallet](connecting-to-a-wallet).
:::

## Mengimpor LSK di Mainnet Lisk

Untuk mengimpor LSK di Mainnet Lisk, lakukan langkah-langkah berikut:

1. Buka alamat kontrak token LSK di Blockscout: [0xac485391EB2d7D88253a7F1eF18C37f4242D1A24](https://blockscout.lisk.com/token/0xac485391EB2d7D88253a7F1eF18C37f4242D1A24).
2. Klik ikon MetaMask.

    <img src="/img/docs/import-lsk/clickMetaMask.png" alt="Getting started"  /> 
3. Jendela pop-up MetaMask akan muncul meminta konfirmasi untuk penambahan Token Kustom.
  <img src="/img/docs/import-lsk/confimAddition.png" alt="Getting started" width="300px" />
4. Klik tombol `Add token`, setelah token berhasil ditambahkan, pesan akan ditampilkan di layar: `Successfully added token to your wallet`.

## Mengimpor LSK di Testnet Lisk Sepolia

Untuk mengimpor LSK di Testnet Lisk Sepolia, lakukan langkah-langkah berikut:

1. Buka alamat kontrak token LSK di Blockscout: [0x8a21CF9Ba08Ae709D64Cb25AfAA951183EC9FF6D](https://sepolia-blockscout.lisk.com/token/0x8a21CF9Ba08Ae709D64Cb25AfAA951183EC9FF6D).
2. Klik ikon MetaMask.

    <img src="/img/docs/import-lsk/clickMetaMask2.png" alt="Getting started"  /> 
3. Jendela pop-up MetaMask akan muncul meminta konfirmasi untuk penambahan Token Kustom.
  <img src="/img/docs/import-lsk/confimAddition.png" alt="Getting started" width="300px" />
4. Klik tombol `Add token`, setelah token berhasil ditambahkan, pesan akan ditampilkan di layar: `Successfully added token to your wallet`.

## Mengimpor LSK di Ethereum Mainnet

Untuk mengimpor LSK di Ethereum Mainnet, lakukan langkah-langkah berikut:

1. Buka alamat kontrak token LSK di Etherscan: [0x6033F7f88332B8db6ad452B7C6D5bB643990aE3f](https://etherscan.io/token/0x6033f7f88332b8db6ad452b7c6d5bb643990ae3f).
2. Klik opsi `Add Token to MetaMask (Web3)` di menu dropdown seperti yang ditunjukkan di bawah ini.

    <img src="/img/docs/import-lsk/clickMetaMask3.png" alt="Getting started"  /> 
3. Jendela pop-up MetaMask akan muncul meminta konfirmasi untuk penambahan Token Kustom.
  <img src="/img/docs/import-lsk/confirmAddition2.png" alt="Getting started" width="300px" />
4. Klik tombol `Add token`, dan token akan ditambahkan ke wallet MetaMask Anda dan ditampilkan dalam daftar token.

:::tip
Untuk mempelajari lebih lanjut tentang semua metode yang tersedia untuk mengimpor token, lihat panduan [Cara Menampilkan Token di MetaMask](https://support.metamask.io/managing-my-tokens/custom-tokens/how-to-display-tokens-in-metamask/) dari MetaMask.
:::
