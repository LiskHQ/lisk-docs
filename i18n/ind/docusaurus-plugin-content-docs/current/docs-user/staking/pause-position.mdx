Panduan ini menjelaskan langkah demi langkah cara jeda hitung mundur pada [posisi penguncian](overview#locking-positions), untuk mendapatkan lebih banyak [kekuatan suara](/governance/overview#voting-power) di Lisk DAO dan menghindari penurunan [reward](overview#staking-rewards) harian.

:::tip
jeda posisi penguncian Anda dapat [meningkatkan kekuatan suara](/governance/overview#boosting-the-voting-power) hingga **200%**.
:::

1. Masuk ke [Dashboard Staking](https://portal.lisk.com/staking/dashboard) di Portal Lisk.
    Hubungkan dompet Anda jika belum dilakukan.
2. Di daftar posisi penguncian Anda, klik posisi penguncian yang ingin Anda tangguhkan untuk memperluasnya.
    <img src={positions} /><br />
3. Klik tombol `Pause Position`.
    Sebuah prompt akan muncul, menginformasikan Anda tentang proses penjedaan.
    <img src={pause} /><br />
    Konfirmasi bahwa Anda memahami konsekuensi dari penjedaan posisi dengan mengklik tombol `Pause Countdown`.
4. Konfirmasi tindakan di dompet Anda.
5. Posisi penguncian Anda sekarang akan dijeda, dan Anda akan melihat pesan konfirmasi.
    <img src={posPaused} /><br />

Jika Anda memuat ulang halaman, Anda seharusnya melihat kekuatan suara yang ditampilkan di dashboard staking telah meningkat sesuai.

:::note
[Delegasi](/governance/overview#delegation) Anda di Lisk DAO akan diperbarui secara otomatis untuk mencerminkan kekuatan suara yang baru.
:::
