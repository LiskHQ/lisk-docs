## Cara staking token LSK Anda

:::info
Anda dapat melakukan staking dengan jumlah token LSK mulai dari minimal 0.01 LSK
:::

1. **Masuk ke halaman `Stake Tokens`** di Lisk Portal: <br />
    → https://portal.lisk.com/staking/stake ← 
2. **Hubungkan dompet Lisk Anda** ke Lisk Portal, jika belum dilakukan.
3. **Pilih jaringan tempat Anda ingin melakukan staking.** 
    Ini harus berupa `Lisk` atau `Lisk Sepolia Testnet`.
4. Sekarang isi formulir sesuai langkah-langkah berikut untuk menyelesaikan proses staking.
    **Masukkan jumlah LSK yang ingin Anda staking.** <br />
    <img src={amount} /><br />
    Anda dapat menggunakan tombol %-di bawah untuk dengan mudah melakukan staking sejumlah persentase dari saldo total LSK Anda.
5. **Tetapkan periode kunci.** <br />
    <img src={duration} /><br />
    Dengan menetapkan periode kunci, Anda menentukan durasi di mana token LSK Anda akan dikunci.
    Selama token tersebut dikunci, mereka tidak dapat digunakan untuk tujuan lain.
6. **Klik tombol `Stake`.**<br />
    <img src={stake} width="80%"/><br />
    Ini akan meminta Anda untuk memulai staking dan menyetujui batas pengeluaran untuk kontrak staking.
7.  **Setujui batas pengeluaran dengan mengklik tombol `Approve`.** <br />
    Anda dapat melewati langkah ini jika sebelumnya sudah menyetujui batas pengeluaran yang lebih tinggi atau sama untuk kontrak staking.<br />
    <img src={approve} width="60%"/><br />
    Konfirmasikan batas pengeluaran untuk kontrak Staking di dompet Anda.
8. **Buat posisi kunci dengan mengklik tombol `Stake`.**<br />
    Setujui transaksi di dompet Anda.

Itu saja! 🎉 Anda telah berhasil staking token LSK Anda.

Seperti yang dapat Anda lihat di [Dashboard Staking](https://portal.lisk.com/staking/dashboard), Anda menerima kekuatan suara di Lisk DAO dan reward staking, yang sebanding dengan jumlah dan durasi kunci yang Anda staking.

Untuk menggunakan kekuatan suara Anda dalam berpartisipasi dalam tata kelola terdesentralisasi Lisk, pastikan untuk [Delegasikan kekuatan suara Anda](/governance/delegation) kepada diri Anda sendiri atau delegasi pilihan Anda.

Setelah setidaknya satu hari melakukan staking, Anda seharusnya melihat jumlah [reward staking](overview#staking-rewards) yang dapat Anda klaim hingga saat itu.
Lanjutkan ke bagian berikutnya untuk mempelajari cara mengklaim reward Anda.

## Cara mengklaim reward staking

Reward staking dapat diklaim kapan saja.
Saat mengklaim, Anda dapat memilih untuk menerima reward tersebut atau men-stake ulang reward tersebut, sehingga meningkatkan jumlah yang terkunci dan meningkatkan kekuatan suara Anda dalam tata kelola.

1. **Masuk ke tab `Claim` pada halaman `Stake Tokens`** di Lisk Portal:<br />
    → https://portal.lisk.com/staking/claim ← <br />
     **Hubungkan dompet Lisk Anda** ke Lisk Portal, jika belum dilakukan.
2. **Pilih jaringan tempat Anda ingin melakukan staking.** 
    Ini harus berupa `Lisk` atau `Lisk Sepolia Testnet`.
3. **Klik tombol `Claim Rewards`.**
4. **Verifikasi reward Anda, siap untuk diklaim.**<br />
    <img src={claimApprove} width="60%"/><br />
5. **Klik tombol `Claim Rewards`.**
6. **Setujui transaksi di dompet Anda** untuk menyelesaikan proses klaim.

## Cara unstake token LSK Anda

Jika Anda ingin unstake LSK yang telah di-stake untuk digunakan untuk tujuan lain, ikuti proses ini:

:::caution
Token LSK hanya dapat di-unstake jika periode kunci dari token yang di-stake telah berakhir.
:::

1. **Masuk ke tab `Unstake` pada halaman `Stake Tokens`** di Lisk Portal: <br /> 
    → https://portal.lisk.com/staking/unstake ← <br />
    **Hubungkan dompet Lisk Anda** ke Lisk Portal, jika belum dilakukan.
2. **Pilih jaringan tempat Anda ingin melakukan unstaking.** 
    Ini harus berupa `Lisk` atau `Lisk Sepolia Testnet`.
3. **Pilih posisi kunci yang telah berakhir** yang ingin Anda unstake dan ikuti proses yang dijelaskan.<br />
    <img src={unstake} width="60%"/><br />
4. Sebuah prompt akan muncul, menunggu Anda untuk mengonfirmasi proses unstaking di dompet Anda. <br />
    <img src={unstakePrompt} width="60%"/><br />
    **Setujui transaksi di dompet Anda** untuk menyelesaikan proses unstaking.
